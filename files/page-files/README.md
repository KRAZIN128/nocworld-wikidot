# NOCWORLD文章文件库必读
这里是[NOCWORLD](//nocworld-wiki-cn.wikidot.com)的官方文件库，**请不要使用Wikidot自带的CDN（wdfiles.com）！！！**
## 使用方法
### 上传
1. **准备文件：** 确定你的文件已经在你的设备中。
2. 复制这个链接，随后按照提示文本修改，随后直接点击访问：`https://github.com/KRAZIN128/nocworld-wikidot/upload/main/files/page-files/[你的页面名在URL里面的样子，如“organism-1”]`
3. 把文件拖动到页面上。或点击“`choose your files`”，随后选择你想要的文件。
4. 点击“`Commit changes…`”按钮，等待完成。

### 使用
你可以把你文件的页面的“`https://github.com/KRAZIN128/nocworld-wikidot/blob/main/`”替换成“`https://cdn.jsdelivr.net/gh/KRAZIN228/nocworld-wikidot@main/`”即可。
