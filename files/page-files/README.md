# NOCWORLD文章文件库必读
这里是[NOCWORLD](//nocworld-wiki-cn.wikidot.com)的官方文件库，**请不要使用Wikidot自带的CDN（wdfiles.com）！！！**
## 使用方法
1. **准备文件：** 确定你的文件已经在你的设备中。
2. **重命名：** 将你的文件重命名为以下格式：`[页面在URL中的名称] - [文件名称.file]`，如`organism-1 - jellyfruit.png`（这样通常较为方便操作）。
3. 点击页面右上角选项卡中的“`Upload files`”，随后选择你要上传的文件。
4. 点击“`Commit changes…`”按钮。
